# Changelog

All notable changes to this project are documented here.
The format follows [Keep a Changelog](https://keepachangelog.com/), and the project
uses [Conventional Commits](https://www.conventionalcommits.org/).

## [Unreleased]

### Added
- **Ability Quiz** mini-game — identify the champion and ability slot (P/Q/W/E/R) from an
  ability icon. 10 rounds, 100% Data Dragon data (no AI). Audio-ready data model for a
  future "guess the sound" mode.
- **Draft Puzzle** mini-game — 5-round match: complete a draft by finding the best missing
  pick. Difficulty + role selection (incl. random), progressive hints, hardcore (no-hints)
  scoring bonus, and a final S/A/B/C/D rank recap.
- Real-data grounding for the puzzle: answer keys validated against OP.GG's role meta
  (filters wrong-role picks), enemy-laner counter win rates injected, and an
  `applyRealDataFloor` safety net so proven lane counters are never mislabeled.
- Multi-provider AI with automatic fallback (Gemini → Groq → OpenRouter).
- Real OP.GG matchup/counter data via the CORS proxy, surfaced in suggestions and used to
  correct the AI.

### Changed
- Extracted scoring/grounding into pure, unit-tested functions.
- Typed the AI chat-response envelope (`AiChatResponse`) — removed most `any` from the AI service.
- CI now also runs the frontend unit-test suite.

### Removed
- Dead code: unused `navbar`, `loading-spinner`, `champion-card` components; the
  `error`/`loading`/`groq` interceptors; and the superseded `GroqHttpService`.
- Trivial autogenerated "should create" spec stubs (replaced with meaningful unit tests).

### Fixed
- Draft-history entries could collide when saved in the same millisecond
  (`Date.now()` id) — now uses a collision-resistant id. (Caught by a new unit test.)
- Champion IDs with spaces/apostrophes (e.g. "Miss Fortune", "K'Sante") broke image URLs —
  now sanitized to Data Dragon format.
